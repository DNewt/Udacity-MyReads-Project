# MyReads Project

MyReads is a project built to organise books into three different categories (Currently Reading, Want to Read and Read) 
The purpose of this project was to learn how to structure a React application.

## TL;DR

To run this project:

* Navigate to your working directory
* Clone the project (git clone)
* install all project dependencies with `npm install`
* start the development server with `npm start`

Note: You will require Node (v6+)

# What this App can do
